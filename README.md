# Domowa biblioteka

Aplikacja webowa umożliwiająca zarządzanie domową biblioteką.

# Wykorzystane technologie i biblioteki:

- Java
- Spring Boot
- Hibernate
- Thymeleaf
- Bootstrap
- Maven
